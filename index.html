      ABOUT THE PROJECT

      I want creatives without a coding background to have a toolkit that can be used to create generative art because it allows creatives to experiment with a variety of visual ideas quickly.


      HOW IT WORKS

      Users can use this tool to "code their creativity" using HTML, CSS, and JavaScript. The goal is for them to gain experience interpreting code while still being creative.
